# Contributing

프로젝트에 기여해 주셔서 감사합니다.

## 기본 원칙

- 작은 단위로 변경해 주세요.
- 변경 이유를 커밋 메시지에 명확히 남겨 주세요.
- 기존 동작을 깨지 않도록 우선 확인해 주세요.

## 로컬 확인

```bash
pnpm type-check
pnpm build
```

필요 시 아래도 함께 확인해 주세요.

```bash
pnpm lint
```

## PR 작성 가이드

- 변경 목적
- 주요 변경 파일
- 테스트/검증 방법
- 영향 범위

## 이슈/문의

- 재현 방법과 기대 결과를 함께 작성해 주세요.
- 로그/스크린샷이 있으면 첨부해 주세요.

감사합니다.
